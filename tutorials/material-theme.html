Installing Material Design theme on Linux Mint

Introduction

Material Design is a visual style created by Google in 2014. If you are obsessed with it, like me, then you'll be happy to know that it is also available as a Linux Mint theme. This tutorial is actually very easy, but I made it step-by-step so it looks harder than it actually is.

Install git

You need git to download the theme. Open a terminal and paste:

sudo apt install git

Tip: Working with apt

When using apt it may ask for your password. Just type it in and press enter. apt may also ask for you to type y or n. Type y and press enter to continue.

Press enter.

Install themes

In the same terminal type: 

git clone https://github.com/snwh/
            paper-icon-theme.git
You are now downloading the Paper icon theme. Open Files. Right-click on an empty space, then check Show hidden files. Open the paper-icon-theme folder. Select the Paper and Paper-Mono-Dark folders, then cut (Ctrl-X) them. Go back using the up arrow. Open the .icons folder. Paste (Ctrl-V) the files here. Open Themes. Click the Add/Remove tab. Search for Adapta. Click the download button on both themes. Click the Themes tab. Change the title bar theme to Adapta. Change the icon theme to Paper. Change the control (window content) theme to Adapta if you want a light theme, or Adapta-Nokto for a dark one. Change the desktop theme to Adapta / Adapta-Nokto. Open the Font Selection menu. Set the default font to Roboto Regular. If you do not have it, run sudo apt install fonts-roboto in a terminal. Repeat this for the desktop font. Set the window title font to Roboto Medium. Now you are done!
